# 文件系统

- 只有一个顶级目录，即：根目录，用 `/` 表示；
- 目录层级使用 `/` 表示，如：`/usr/local/temp`;

## cd 命令

切换当前目录至某一个目录，英文：Change Directory；

### 命令格式

`cd [目录名]`

### 常见范例

- `cd /home/temp`，切换到 /home/temp 目录；
- `cd /`，切换到根目录；
- `cd ..`，返回上级目录；
- `cd ~`，切换到当前用户目录；
- `cd -`，返回进入此目录之前所在的目录；
- `cd !$`，把上个命令的参数作为 cd 参数使用；

## pwd 命令

用来查看”当前工作目录“的完整路径，英文：Print Work Directory；

### 命令格式

`pwd [选项]`

### 常用参数

一般情况下不带任何参数，但如果目录是链接时：格式：`pwd -P` 显示出实际路径，而非使用连接(link)路径。

## ls 命令

ls 命令作用是列出目录下的内容

### 命令格式

`ls [选项] [目录路径]`

### 常用参数：

- `-a`，`-all` 列出目录下的所有文件，包括以 `.` 开头的隐含文件；
- `-l`，除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来；
- `-h`，`–human-readable` 以容易理解的格式列出文件大小 (例如 1K 234M 2G)

## mkdir 命令

用来创建指定名称的目录，要求创建目录的用户在当前目录中具体写权限，并且指定的目录名不能与当前目录下的目录重名，即 make directory 缩写。

### 命令格式

`mkdir [选项] 目录`

### 选项参数

- `-m`，`--mode=模式`，设定权限 `<模式>` 类似 chmod；
- `-p`，`--parents` 可以是一个路径名称。若路径中的目录不存在，系统会自动创建对应的目录，适用于创建连续多层级的目录；
- `-v`，`--verbose` 每次创建目录都会显示信息；

### 命令示例

```shell
## 创建目录 temp
mkdir temp

## 递归创建多个目录，并显示创建信息
mkdir -vp temp/logs

## 创建权限为 777 的目录
mkdir -m 777 temp

## 命令创建项目的目录结构
mkdir -vp project/{docs,src/{views,assets},public/static,dist}
```

## rm 命令

删除一个目录中的一个或多个文件或目录，如果没有使用 `-r` 选项，则 `rm` 不会删除目录。如果使用 `rm` 来删除文件，通常仍可以将该文件恢复原状。

### 命令格式

`rm [选项] 目录/文件`

### 选项参数

- `-f`，`--force` 忽略不存在的文件，从不给出提示；
- `-i`，`--interactive` 进行交互式删除；
- `-r`，`-R`，`--recursive` 递归删除目录和子目录；
- `-v`，`--verbose` 详细显示进行的步骤；

### 命令示例

```shell
## 删除文件
rm temp.md

## 删除目录
rm -r temp

## 会将 test3 子目录及子目录中所有档案删除,并且不用逐个确认
rm -rf test3

## 删除以 -f 开头的文件
rm -- -f
```

## tail 命令

用于显示指定的文件末尾内容，不指定文件时，作为输入信息进行处理。常用查看日志文件。

### 命令格式

`tail [必要参数] [选择参数] [文件]`

### 选项参数

- `-f`，表示持续跟踪；
- `-n`，查看行数，不填默认 10 行；

### 命令示例

```shell
## 持续跟踪查看日志
tail -f temp.log

## 从第十行开始显示文件
tail -n +10 temp.log
```

## find 命令

在目录结构中搜索文件

### 命令格式

`find <目录路径> -options [查找内容]`

### 选项参数

- `-name` 按照文件名查找文件，支持通配符 `*`，如：`*test*`；
- `-size` 按照文件大小查找，语法：`find 目录路径 -size +|-n[kMG]`；
  - `+n`，表示文件大小大于 n；
  - `-n`，表示文件大小小于 n；
  - `kMG`，表示大小单位，k：（小写）表示 KB，M 表示 MB，G 表示 GB；
- `-mtime`, 按照文件的更改时间来查找文件，语法：`find 目录路径 -mtime +|-n`；
  - `-n`，表示文件更改时间距现在 n 天以内，
  - `+n`, 表示文件更改时间距现在 n 天以前。

### 命令示例

```shell
## 在当前目录中查找文件名包含 linux 的文件
find . -name "*linux*"

## 在当前目录中查找大于 500KB 的文件
find ./ -size +500k

## 配合管道符号 + wc 命令直接输出结果数量
find ./ -size +500k | wc -l

## 在当前目录中查找更改时间在10天前的文件数量
find ./ -mtime +10 | wc -l
```

## 统计 wc 命令

wc（表示：Word Count）命令的功能为统计指定文件中的字节数、字数、行数，并将统计结果显示输出。

### 命令格式

`wc [-options] 文件`

### 选项参数

- `-c`，统计字节数；
- `-l`，统计行数；
- `-m`，统计字符数，这个选项不能与 `-c` 一起使用；
- `-w`，统计字数，一个字被定义为有空白、跳格、换行字符分割的字符串；

### 命令示例

- 查看文件的字节数、字数、行数

`wc temp.log`

- 用 wc 命令怎么做到只打印统计数字不打印文件名

`cat file2.txt | wc -l`

- 用来统计当前目录下的文件数

`ls -l | ws -l`
